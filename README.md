# MediaGuard 🔍

**MediaGuard** is a terminal-based forensic toolkit built by SV 🕶️  
Helps detect AI-generated images, hidden metadata, and tampering.

## 🔧 Features
- Image hash verification
- Metadata extraction
- Face detection
- Error Level Analysis (ELA)
- AI-generation detection
